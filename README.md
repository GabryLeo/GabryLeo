## ⭐ Projetos em Destaque

### 🎫 FlowHelpDesk — Sistema Full-Stack de Gerenciamento de Tickets
**Tech Stack:** TypeScript • Express • React • PostgreSQL • Prisma
Sistema profissional para gerenciar tickets de suporte técnico com dashboard, autenticação JWT e notificações em tempo real.
🔗 [Repositório](https://github.com/GabryLeo/FlowHelpDesk) | 🌐 [Deploy](https://flow-help-desk.vercel.app)

### 💰 Finanças do Casal — Gestor Financeiro Moderno
**Tech Stack:** JavaScript • Chart.js • Google Apps Script
App responsivo para controlar finanças em casal com gráficos inteligentes, lançamentos com parcelas e tema claro/escuro.
🔗 [Repositório](https://github.com/GabryLeo/financas-casal)

### 🤖 Promo-es-Perfumes — Bot de Monitoramento Inteligente
**Tech Stack:** Python • Web-Scraping • Automação • WhatsApp
Bot que rastreia promoções de perfumes na web e envia alertas automáticos via WhatsApp.
🔗 [Repositório](https://github.com/GabryLeo/promo-es-perfumes-)

### ⚽ Copa do Mundo 2026 — Bot de Notificações
**Tech Stack:** Python • Playwright • ESPN API
Bot que monitora jogos ao vivo e envia resultados com prints via WhatsApp com intervalo dinâmico de checagem.
🔗 [Repositório](https://github.com/GabryLeo/copa-do-mundo)
